# How to Use

Welcome to your website's documentation. Every part of the site is documented in
detail to give you an understanding of how to manage and update its content.

**Important**: When reading the documentation, be sure to click the **Docs**
button toward the top of the page to see the correct information.

## Sections

Use the sidebar to the left to navigate through the different sections of the
documentation.

#### Documentation

General information about website content, its underlying services, and
management guides.

- **General**: Overview of managing the site and its content.
- **Prismic CMS**: Guides on using the site's content management system,
  [Prismic](http://prismic.io/).
- **Workflows**: Instructions on performing common content updates.

#### Pages

All pages on the site are managed in Prismic using Page documents. To learn more
about the fields and Slices available to Pages, see the documentation in the
**Pages** section.

#### Document Types

While Pages are the primary document type in Prismic to manage content, all
other content is managed through a collection of other document types. This
includes content such as the site's copyright text and navigation menus. Learn
more about the different fields available to each document type in their
respective documentation within the **Document Types** section.

#### Style Guide

Colors and typography used throughout the site are listed within the **Style
Guide** section. Use these specifications when creating new website content.

#### Components

The low-level elements that make up the site are listed within the
**Components** section. This is primarily for future designers and developers
working on the website, but may be referenced in future website development
discussions.
