# Services

All services used to run and maintain the site.

## Overview

The site uses several services to host, store, and track website content and
visitors. Some require a monthly or annual service fee.

## Services

### General

- [**Netlify**](https://netlify.com): Static site hosting platform used to serve
  and build the site.

- [**Prismic**](https://prismic.io): Content management system used to edit
  content on the site.

### User Interface

- [**Netlify Forms**](https://www.netlify.com/products/forms/): Static site
  hosting platform used to accept form submissions.

- [**MapBox**](https://www.mapbox.com): Embeddable interactive map used to
  visualize projects and closures. (Free within limits)

### Analytics

- [**Google Tag Manager**](https://marketingplatform.google.com/about/tag-manager/):
  Website tag manager used to load Google Analytics and any other tracking
  services. (Free)

- [**Google Analytics**](https://marketingplatform.google.com/about/analytics/):
  Website analytics platform used to understand website visitors. (Free)
